# Vaulty-vault
fast and reliable implementation of vault using go.



## Deployment

make executable and run dir.sh

```bash
chmod +x ./dir.sh

./dir.sh
```

after retrieving data change run.sh

```bash
export VAULT_TOKEN=<VAULT_TOKEN>
```

make executable and run run.sh

```bash
chmod +x ./dir.sh

./dir.sh
```
